# 前端预览服务设计

## 1. 概述

### 1.1 项目背景
为 Daily News Pro 创建前端预览服务，供普通公众用户查看新闻文章。

### 1.2 设计目标
- 公开访问，无需登录
- 清晰的新闻浏览体验
- 支持多维度筛选和搜索
- 干净的阅读视图

---

## 2. 页面结构

### 2.1 路由设计

| 路径 | 页面 | 说明 |
|------|------|------|
| `/preview` | 预览首页 | 新闻列表 + 筛选搜索 |
| `/preview/article/:id` | 文章详情页 | 阅读视图 + 原文跳转 |

### 2.2 页面关系
```
/preview (列表页)
    └── /preview/article/:id (详情页)
```

## 3. 功能设计

### 3.1 预览首页

#### 3.1.1 页面布局
```
┌─────────────────────────────────────────┐
│  Logo      [搜索框]           [深色模式] │  <- Header
├─────────────────────────────────────────┤
│  全部 | 36kr | ProductHunt | ...        │  <- 来源 Tabs
├─────────────────────────────────────────┤
│  [全部] [当天] [本周] [当月]            │  <- 时间筛选
├─────────────────────────────────────────┤
│  [标签1] [标签2] [标签3] ...           │  <- 标签筛选（可多选）
├─────────────────────────────────────────┤
│  ┌─────┐  ┌─────┐  ┌─────┐            │
│  │ Card│  │ Card│  │ Card│            │
│  └─────┘  └─────┘  └─────┘            │
│              [加载中...]                │  <- 无限滚动
└─────────────────────────────────────────┘
```

#### 3.1.2 筛选功能

| 筛选维度 | 类型 | 说明 |
|----------|------|------|
| 来源 | 单选 | 全部 / 36kr / ProductHunt / ... |
| 时间 | 单选 | 全部 / 当天 / 本周 / 当月 |
| 标签 | 多选 | 选中多个标签时为 AND 关系 |
| 关键词 | 文本 | 搜索标题和摘要 |

#### 3.1.3 新闻卡片内容
```
┌─────────────────────────┐
│  [封面图]                │
│  来源名称 · 2小时前      │
│  文章标题                │
│  文章摘要...             │
│  [标签1] [标签2]        │
└─────────────────────────┘
```

#### 3.1.4 默认状态
- 默认显示当天抓取的新闻
- 时间筛选默认选中"全部"

## 4. 技术方案

### 4.1 技术栈
与现有前端保持一致：
- React 18 + TypeScript
- Vite 5
- Ant Design 5
- React Router DOM 6

### 4.2 部署方案
方案 1：管理后台添加登录验证
- 预览页面公开访问
- 管理后台需要登录认证

### 4.3 项目结构
```
frontend/src/pages/Preview/
├── index.tsx           # 预览首页
├── ArticleDetail.tsx   # 文章详情页
├── components/
│   ├── Header.tsx      # 顶部导航栏
│   ├── SourceTabs.tsx  # 来源 Tab 切换
│   ├── TimeFilter.tsx  # 时间筛选
│   ├── TagFilter.tsx   # 标签筛选
│   ├── SearchBar.tsx   # 搜索输入框
│   ├── NewsCard.tsx    # 新闻卡片
│   ├── NewsList.tsx    # 无限滚动列表
│   └── ArticleContent.tsx # 文章正文渲染
├── hooks/
│   └── useArticles.ts  # 文章数据 hook
└── context/
    └── FilterContext.tsx # 筛选状态管理
```

## 7. 状态管理

### 7.1 筛选状态

```typescript
interface PreviewFilter {
  source: string;      // 来源（空=全部）
  timeRange: '' | 'today' | 'week' | 'month';  // 时间为空=全部
  tags: string[];       // 选中标签（空=全部）
  keyword: string;      // 搜索关键词
}
```

### 7.2 主题状态
使用 Ant Design 的 ConfigProvider 管理深色模式，通过 React Context 切换。

---

## 8. 设计决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 渲染方式 | CSR | 与现有技术栈一致，改动最小 |
| 加载方式 | 无限滚动 | 用户体验更流畅 |
| 搜索方式 | 关键词+筛选组合 | 支持高级搜索需求 |
| 时间筛选默认值 | 全部 | 默认展示所有新闻 |
